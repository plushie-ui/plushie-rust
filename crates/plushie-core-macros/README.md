# plushie-core-macros

Procedural macros for [Plushie](https://github.com/plushie-ui/plushie-rust)
types and widgets. **Pre-1.0**

Provides derive macros used by `plushie-core` and the wider Plushie
crate ecosystem. Not intended for direct use; depend on `plushie-core`
or `plushie-widget-sdk` instead, which re-export the macros.
